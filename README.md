# JSwallet
nothing serious here...

## Run
Create `wallet.json` file and put your mnemonic phrase and RPC url into this file.
```json
{
    "networks": [
        {
            "rpcUrl": "",
            "mnemonicPhrase": "",
            "address": "",
            "tokens": [
                ""
            ]
        }
    ]
}

```

Run:
```sh
npm i ethers
node index.js
```
